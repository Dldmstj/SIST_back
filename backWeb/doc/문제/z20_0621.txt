[1단계:개념] 1. session scope의 4가지 범위를 조원들과 함께 이해가 되었는지 확인하세요,
	- page Scope : 현재 페이지에서만 유지되는 범위. 해당 페이지에서만 변수에 접근할 수 있다.
	- Request Scope : 현재 요청에서만 유지되는 범위. 한 번의 요청에서만 변수에 접근할 수 있다.
	- Session Scope : 사용자의 세션에 속한 모든 페이지에서 유지되는 범위. 해당 세션 내의 모든 페이지에서 변수에 접근할 수 있다.
						같은 브라우저를 사용할 때. 모든 브라우저 창을 닫고 시작하지 않는 이상 유지
	- Application Scope : 웹 애플리케이션 전역에서 유지되는 범위. 애플리케이션의 모든 페이지에서 변수에 접근할 수 있다.
						다른 브라우저를 사용하더라도 유지. 모든 브라우저 창을 닫고 다시 실행하더라도 유지.
						단 tomcat 재기동시 사라진다.
